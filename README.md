## 🏗️ Technical Architecture

### Microservices Stack
*   **Auth Service**: Identity management and Sovereign Shield RBAC.
*   **Data Science Service**: ML training, telemetry, and code execution.
*   **Actuarial Service**: Projections, simulations, and pricing engines.
*   **Accounting Service**: Enterprise financial ledger and reporting.
*   **Compliance Service**: IFRS 17 & Solvency II regulatory logic.
*   **AI Service**: Reasoning-based Intelligence Copilot and Knowledge RAG.

### Infrastructure
*   **Frontend**: React 18 + Vite + Tailwind + Glassmorphism Digital Interface.
*   **Data Tier**: PostgreSQL (TimeScaleDB) + Redis + Kafka Event Streaming.
*   **Orchestration**: Docker Compose with health-layered microservices.
